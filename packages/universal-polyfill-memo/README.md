# `universal-polyfill-memo`

> Memo polyfill for React/Rax

## Usage

```js
import Engine from 'universal-polyfill-engine';
import memo from 'universal-polyfill-memo';

import React from 'react';
Engine.set(React);

import * as Rax from 'rax';
Engine.set(Rax);
```
